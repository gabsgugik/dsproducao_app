# API Previsão de Vendas nas Próximas 6 semanas utilizando Machine Learning
API desenvolvida para o projeto de Previsão de compras nas próximas 6 semanas utilizando Machine Learning.

A API foi desenvolvida utilizando Python e microframework Flask, seguindo os passos abaixo:

1. A entrada consiste de um dicionário no formato JSON contendo os dados;

2. Os dados dados são limpos e tratados;

3. Para cada registro é realizada a previsão de vendas da loja nas próximas 6 semanas, e o resultado armazenado
em uma nova coluna chamada 'prediction';

4. A nova coluna 'prediction' é retornada junto com os dados originais no formato dicionário JSON.
